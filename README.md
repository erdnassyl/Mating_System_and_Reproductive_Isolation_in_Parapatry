# MSRIP_TL_BM_project
Mating System and Reproductive Isolation in Parapatry, Two Locus with BDMi. Internship Project supervised by Sylvain Glémin and Lucas Marie-Orleach. 

C++ codes for simulations. 
MSRIP_TL_MyFunctions.h stores every functions needed for the Life Cycle Model used in the project. 
MSRIP_TL_BM_SC_Main.cpp stores the simulations for Secondary Contact after allopatric phase in Continent-Island case. 
MSRIP_TL_BM_CI_Main.cpp stores the simulations for the build up of Reproductive Isolation in Continent-Island case. 
MSRIP_TL_BM_SC_Parallel.sh is the bash code associated to MSRIP_TL_BM_SC_Main.cpp with parameters scales. 
MSRIP_TL_BM_CI_Parallel.sh is the bash code associated to MSRIP_TL_BM_CI_Main.cpp with parameters scales.
